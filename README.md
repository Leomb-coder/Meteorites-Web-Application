# [Asteroids-Web-Application](https://asteroids-web-application.onrender.com)
Web Application made to manage and analize near Earth Asteroid information using [NASA's API Asteroids - NeoWs](https://api.nasa.gov/)<br>
Currently hosted on Render, but limited by free plan;

This project was made for me to learn more about Web Applications and Hosting, and im really into Astronomy, so thats where the idea came from --
This is an Open-Source repository so feel free to help with any issues we find.

## Features
- Auto-fetch asteroid data from NASA API
- Save API data into DB (favorites)
- CRUD operations
- Handle errors (API down, bad response)
- Filter (by date, size, etc.)
- User authentication (login/register)
- Dashboard with information

## Target Audience
- Personal project
- People who want to study or are curious about the recognition of fragments of celestial bodies.
